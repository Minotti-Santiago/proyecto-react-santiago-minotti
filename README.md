#Santiago Minotti

En este proyecto he creado un e-commerce en el cual vendo periféricos para computadoras. El mismo cuenta con varios productos como, teclados, mouse y mousepad, también puedes ver más información sobre el producto y colocar la cantidad q deseas comprar, además, puedes agregarlos al carrito, ver el monto total de lo que compras, y realizar la compra colocando tus datos personales.

En el proyecto se utilizó, react-router-rom, useEffect, useState, Firebase, Firestore, base de datos( donde guardamos los productos y los datos de la compra del cliente).
